# CodeAlpha_CreditScoring

## Credit Scoring Model for CodeAlpha Internship

### 📋 Project Overview
This project builds a machine learning model to predict creditworthiness using the German Credit Dataset.

### 📊 Dataset
- **Source:** German Credit Dataset (UCI Machine Learning Repository)
- **Instances:** 1000
- **Features:** 20
- **Target:** Good credit (1) vs Bad credit (2)

### 🧠 Models Implemented
- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier

### 📈 Performance Metrics
| Model | Accuracy | Precision | Recall | F1-Score | ROC-AUC |
|-------|----------|-----------|--------|----------|---------|
| Logistic Regression | 0.765 | 0.627 | 0.533 | 0.577 | 0.807 |
| Decision Tree | 0.650 | 0.422 | 0.450 | 0.435 | 0.593 |
| Random Forest | 0.750 | 0.647 | 0.367 | 0.468 | 0.800 |

### 📁 Files in this Repository
- `german_credit.data` - The dataset
- `credit_scoring_model.pkl` - Trained Random Forest model
- `model_results.csv` - Performance comparison of all models
- `Credit_Scoring_Task1.ipynb` - Complete code with analysis
